# Security

This repository is a local evaluation demo. It does not require API keys, external model calls, or private datasets.

## Supported Use

- Local JSONL evaluation datasets.
- Local rubric configs.
- Generated Markdown and JSON reports.

## Data Handling

Do not commit:

- Private prompts or user data.
- Customer conversations.
- API keys or tokens.
- Proprietary evaluation guidelines.

Use synthetic or redacted examples for public demos.

## Reporting Issues

If you find a security issue, open a private report through GitHub Security Advisories if available. If that is not available, contact the repository owner directly and avoid posting sensitive details in a public issue.
